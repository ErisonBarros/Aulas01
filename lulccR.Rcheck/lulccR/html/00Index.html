allocate Allocate land use change spatially Perform spatially explicit allocation of land use change using different methods. Currently the function provides an algorithm based on the CLUE-S model (Verburg et al., 2002) and a novel stochastic procedure. 'allocate' is designed to be easily extensible. To write a new allocation method (apart from writing the algorithm itself) one simply needs to create a new class inheriting from 'ModelInput' and an 'allocate' method for this class.
